# Automate Catalog Processing Handoff

This project contains scripts to parse a WhatsApp chat export to build a product catalog and use OCR to extract product codes from associated images and videos.

## Project Structure

- **`process_chat.py`**: Parses exported WhatsApp chat logs. It extracts product descriptions, original prices, calculates a 35% margin price, and dynamically generates new descriptions with the updated price.
- **`extract_codes.py`**: Reads the catalog generated by `process_chat.py` and uses EasyOCR to scan the corresponding media files (images or middle frames of videos) from the `Data/` directory. It filters out common spam words and extracts alphanumeric product codes, appending them to the catalog.
- **`Data/`**: A directory that should contain the exported WhatsApp chat `.txt` file along with the exported media files (e.g., `IMG-***.jpg`, `VID-***.mp4`).
- **`Processed_Catalog.csv` / `.xlsx`**: The generated outputs containing the combined catalog data.

## Getting Started

### 1. Prerequisites
Ensure you have Python 3.x installed on your system. 

### 2. Install Dependencies
Install the required packages using the generated `requirements.txt`.
```bash
pip install -r requirements.txt
```
*Note: `extract_codes.py` uses `easyocr` with `gpu=False`. CPU-only PyTorch may be required depending on your local setup to avoid heavy GPU dependencies.*

### 3. Setup Data
Place your exported WhatsApp chat text file and all corresponding media files into the `Data/` folder.
*(Note: `process_chat.py` currently hardcodes the chat filename to `WhatsApp Chat with SuruArts Creations.💃🏻😍.txt`. You may need to update this filename in `process_chat.py` line 5 to match your specific chat export).*

### 4. Running the Pipeline

**Step A: Process the Chat log**
Run `process_chat.py` to parse the chat text and create the initial catalog outline.
```bash
python process_chat.py
```
This will output `Processed_Catalog.csv` and `Processed_Catalog.xlsx`.

**Step B: Extract OCR Codes**
Run `extract_codes.py` to read the newly created catalog, perform OCR on each media file associated with a product, and find the product code.
```bash
python extract_codes.py
```
This updates both `Processed_Catalog.csv` and `Processed_Catalog.xlsx` with an additional column `Extracted Codes (OCR)`.
